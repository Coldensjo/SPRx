# Tauri Icons

This directory should contain the following icon files:

- `32x32.png` - 32x32 PNG icon
- `128x128.png` - 128x128 PNG icon  
- `128x128@2x.png` - 256x256 PNG icon (for retina displays)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

You can generate these icons from a single source image using tools like:
- [Tauri Icon Generator](https://github.com/tauri-apps/tauri-icon)
- Online icon converters
- Image editing software

For now, placeholder icons will be generated automatically when you run `bun run tauri:dev` for the first time.
